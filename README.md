# VOLTWISE — Smart Energy & Charging

A multi-page product showcase e-commerce demo built with pure HTML, CSS, and JavaScript.
Created for MIS343 — Yanbu Industrial College.

---

## Project Structure

```
voltwise/
├── index.html      ← Homepage (hero, metrics, brand statement)
├── products.html   ← Product catalog
├── product.html    ← Product detail + cart + checkout
├── style.css       ← All styles and design tokens
├── script.js       ← Cart, checkout, gallery, tabs
├── imgs/           ← Product imagery
└── README.md
```

---

## Features

- **Homepage** — Scroll-reveal sections, hero orbs, live-stat counter, metrics marquee, "How it works", brand statement
- **Product Catalog** — Card grid with pricing
- **Product Gallery** — Main image with clickable thumbnails
- **Add to Cart** — Quantity stepper and animated cart button
- **Cart Drawer** — Slide-in drawer with item management and order summary
- **Checkout Panel** — Full form with Saudi regions, delivery options, and payment methods
- **Order Confirmation** — Animated checkmark with order details
- **Tabs** — Description, Specifications, and Reviews sections
- **Responsive Design** — Mobile-tuned typography and spacing across all pages

## Payment Methods Supported

Mada · Visa · Mastercard · Apple Pay · STC Pay · Cash on Delivery

---

## How to Run

1. Open `index.html` in any modern browser — no server required
2. All files must remain in the same folder structure
